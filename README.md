## FR — À propos

Ce projet explore les bases du **trading algorithmique** à l’aide de **Python**, en combinant l’analyse de données financières, des **indicateurs techniques** et des approches de **machine learning**.

L’objectif est de concevoir, tester et évaluer des stratégies quantitatives sur des données de marché réelles, tout en mettant l’accent sur la compréhension des modèles, la performance et la gestion du risque.

### Technologies
- Python  
- Pandas, NumPy  
- Matplotlib  
- Indicateurs techniques (RSI, moyennes mobiles, etc.)  
- Données financières historiques

## ENG — About

This project explores the fundamentals of **algorithmic trading** using **Python**, combining financial data analysis, **technical indicators**, and **machine learning approaches**.

The goal is to design, test, and evaluate quantitative trading strategies on real market data, with a focus on model understanding, performance evaluation, and risk awareness.

### Technologies
- Python  
- Pandas, NumPy  
- Matplotlib  
- Technical indicators (RSI, moving averages, etc.)  
- Historical financial data
